# 彩虹工具 RainbowUtility

## 此插件允许
- 禁用末影螨仇恨
- 禁用受重力影响的方块更新
- 高亮提醒玩家
- 快捷传送点
- 清理聊天框
- 自定义客户端天气与时间
- 丢弃保护
- 向在线管理员获取帮助
- 保护农田
- 向服务器留言反馈
- 管理世界边界

## 前置
- 无

## 兼容性
- Minecraft Spigot 1.13 或更高版本。
- 在 Bukkit+Forge 与 Bukkit+Fabric 混合服务端运行时，自定义客户端天气与时间模块不可用。
